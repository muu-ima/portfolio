## Positioning

それぞれのプロジェクトは、見た目の完成度だけでなく、裏側の設計や業務の読み解きに価値があります。

- tools-hub: 越境ECの利益計算と発送情報管理
- CardCraft: 名刺デザイン編集と書き出し
- nfc-redirect: 配布後も更新できる NFC / QR リンク管理
- kobutsu-ledger-system: 古物台帳とEC販売をつなぐ業務システム
