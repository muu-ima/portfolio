# CardCraft

## Overview

CardCraft は、ブラウザ上で名刺デザインを作成し、表裏プレビュー、保存、画像書き出しまで行える Web エディタです。

テンプレートを選び、文字や画像を配置しながら、編集中に何度でもデザインを確認できます。単体の名刺編集ツールとしてだけでなく、nfc-redirect と組み合わせて、配布後もプロフィールや案内先を更新できる名刺運用を想定していました。

## Main Features

- テンプレート選択
- テキスト追加、編集
- フォントサイズ調整
- テキスト幅調整
- 背景画像の選択
- 自分で用意した画像のアップロード
- 画像配置、サイズ調整
- レイヤー順の管理
- 表面、裏面の切り替え
- プレビュー表示
- PNG / JPEG 書き出し
- ローカルストレージへの保存
- 編集途中のデザイン再開

## Editor Experience

編集画面では、キャンバスを見ながら左側のパネルでテキスト、背景、画像、レイヤーを切り替えて調整できます。

PC ではキャンバスと編集パネルを同時に確認しやすく、モバイルでは限られた幅の中でもプレビューと編集を行き来できる構成にしています。作業を途中で切り上げても、ローカルストレージに保存した内容から再開できるようにしています。

## Difficult Points

- 自由に編集できることと、名刺として破綻しないことの両立
- プレビュー、キャンバス編集、PNG / JPEG 書き出しで同じデザイン状態を同期すること
- キャンバスの拡大縮小時にも、テキストや画像の編集位置がずれないようにすること
- テキスト、画像、背景、レイヤーの状態管理
- 自分でアップロードした画像を、配置やサイズ調整の対象として扱うこと
- PC、タブレット、モバイルで編集できる UI にすること
- 作業を切り上げても再開できる保存設計
- 決済システムと接続する直前まで、商品化を想定していたこと

## Good Points

- ブラウザ上で名刺編集、確認、書き出しまで完結できる
- プレビューと編集画面の状態を同期し、入稿前の確認に近い体験を作れている
- キャンバスの拡大縮小をしても、編集対象の位置関係が崩れにくい
- テキスト、画像、背景、レイヤーを分けて扱い、エディタとして自然な構成になっている
- 自分で用意した画像をアップロードし、名刺デザインの素材として使える
- ローカルストレージ保存により、作業を中断しても再開できる
- PC、タブレット、モバイルを想定した編集 UI まで踏み込んでいる
- nfc-redirect と組み合わせることで、名刺作成から配布後の運用までつながる構想になっている

## Relationship With nfc-redirect

CardCraft で作った名刺に NFC / QR を組み込み、nfc-redirect 側でプロフィールやリンク先を後から変更できるようにする想定でした。

CardCraft が「名刺デザインと入稿データを作る場所」、nfc-redirect が「配布後のリンク先を管理する場所」という役割です。
